# 5. Main Function  

* 여행 예산짜기
* 카테고리(숙박, 식사, 간식 및 주류, 쇼핑, 교통, 액티비티)별 항목(숙소 등급, 식당 등급 등) 추가하기
* 여행별 일정추가 및 경비계산
* 회원가입 / 로그인 
* 카테고리별 지출내역 작성 및 관리
